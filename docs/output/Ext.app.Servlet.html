Class Ext.app.Servlet

Package:Ext.app
Defined In:Servlet.js
Class:Servlet
Extends:Object
A servlet is a HTTP service for serving static resouces. To be used via: Ext.servlet({ // Configuration goes here }) Each instance registers by it's name attribute and is fetchable by: var servlet = Ext.getServlet('nameOfTheServlet');
