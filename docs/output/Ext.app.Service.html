Class Ext.app.Service

Package:Ext.app
Defined In:Service.js
Class:Service
Extends:Object
Represents a service. To be used via: Ext.service({ // Configuration goes here }) Each instance registers by it's name attribute and is fetchable by: var service = Ext.getService('nameOfTheService');
